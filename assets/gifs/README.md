## Creating the GIFs

### Tools you can use:
1. **Screen Recording:**
   - iOS: QuickTime Player or Screen Recording
   - Android: Built-in screen recorder
   - Mac: QuickTime Player or ScreenFlow

2. **GIF Creation:**
   - [GIPHY Capture](https://giphy.com/apps/giphycapture) (Mac)
   - [LICEcap](https://www.cockos.com/licecap/) (Cross-platform)
   - [Kap](https://getkap.co/) (Mac)
   - [FFmpeg](https://ffmpeg.org/) (Command line)

3. **GIF Optimization:**
   - [EZGIF](https://ezgif.com/) - Online GIF optimizer
   - [ImageOptim](https://imageoptim.com/) - Mac app
   - FFmpeg for compression

### Tips:
- Keep GIFs under 2MB for fast loading
- Use 2-5 second loops
- Optimize colors (256 colors max)
- Reduce frame rate if needed (10-15 fps)
- Crop to focus on important areas
